obiHomoloGene Translation of homolog genes

obiHomoloGene is an interface to HomoloGene.

Functions

all_genes(taxid)
Return a set of all genes for organism with taxid
homologs(genename, taxid)
Return a list of homologs (homolog_taxid, homolog_genename) for a homolog group that genename, taxid belong to
homolog(genename, taxid, homolotaxid)
Return a homolog of genename, taxid in organism with holomotaxid or None if homolog does not exist.

Examples

Mapping homologs from yeast to human (obiHomoloGene-yeast-to-human.py)

import orange import obiHomoloGene, obiTaxonomy, obiGene data = orange.ExampleTable("../../../../doc/datasets/brown-selected") geneinfo = obiGene.NCBIGeneInfo('4932') genes = [str(ex["gene"]) for ex in data] for gene in genes: mappedgene = obiHomoloGene.homolog(geneinfo(gene).symbol, '4932', '9606') print gene, mappedgene
