obiTaxonomy: NCBI Taxonomy

obiTaxonomy is a library used for matching organism to NCBI Taxonomy ids and was created for organism name unification across different modules.

Functions

name(taxid)
Return the scientific name for organism with taxid.
other_names(taxid)
Return a list of (name, name_type) tuples but exclude the scientific name.
search(string, onlySpecies=True)
Search the NCBI taxonomy database for an organism by string. If onlySpecies is True then search only in the species and subspecies nodes ob the taxonomy.
lineage(taxid)
Return a list of taxids ordered from the topmost node (root) to taxid.
to_taxid(code)
See if the code is a valid code in any database and return its taxid.

Other modules

For full unification, each module (e.g. obiKEGG, obiGO, ...) should provide the following interface

from_taxid(taxid)
Convert taxid to module's internal organism code.
to_taxid(organism)
Convert module's internal organism code to taxid.
organisms()
Returns a list of tuples (taxid, internal_name)
