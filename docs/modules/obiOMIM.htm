obiOMIM Online Mendelian Inheritance in Man

obiOMIM provides an interface to OMIM - Online Mendelian Inheritance in Man database. For now it only supports mapping genes to diseases.

Functions

diseases()
Return all disease descriptors
genes()
Return a set of all genes referenced in OMIM
disease_genes(disease)
Return a set of all genes referenced by disease in OMIM
gene_diseases()
Return a dictionary {gene: set(disease_objects for gene), ...}

Example

Create a network of connected diseases and save it in pajek .net format(part of obiOMIM-disease-network.py)

import obiOMIM diseases = obiOMIM.diseases() genes = [obiOMIM.disease_genes(disease) for disease in diseases] vertices = [] edges = [] for i in range(len(diseases)): vertices.append('%i "%s"\n' % (i + 1, diseases[i].name)) for j in range(i + 1, len(diseases)): if set(genes[i]).intersection(genes[j]): edges.append('%i %i %i\n' %(i + 1, j + 1, len(set(genes[i]).intersection(genes[j])))) file = open("disease.net", "wb") file.write("*Vertices %i 2\n" % len(vertices)) file.writelines(vertices) file.write("*Edges\n") file.writelines(edges)
